Page rank algorithm 101

  • Keyword visible on page => get ranked
    • Use "DMN" in a sentence on drools.org
    • Then give it time
  • More external links to url/domain => higher ranking
    • Write articles (voxxed, dzone, RHD) or blogs
    • Record YouTube video's
    • Events (JUG, ...)
    • Academic papers
  • 1000 other tips, see YouTube "Backlinko" video's
